## How to start?


1) `git clone https://github.com/mYYcroSoft/network-tool.git`

2) `cd web`

3) `php -S localhost:8000`

## Example

**Input:** 192.168.0.1/15

- IP: 192.168.0.1
- Prefix: 15
- IP validace: Ano
- Třída: C
- Octet: 168
- Net(str): 192.168.0.1
- Net(int): 3232235521
- Binary: 12539a1
- Private flag: Ano
- Broadcast adresa: 192.169.255.255
- Síťová maska: 255.254.0.0
- Síťová adresa: 192.168.0.0
